Remorci agricole cu şnec pentru cereale

Remorcile agricole Chironna (Italia) cuprind o gamă completă de modele cu un bun raport calitate / preţ. Printre acestea se numără şi remorcile cu rabatare hidraulică trilaterală şi cu şnec posterior pentru descărcat cerealele.

Cel mai bine vândut model are următoarele caracteristici tehnice:

✓ Volum 15 mc
✓ Rabatare hidraulică trilaterală
✓ Dimensiuni (L x l x H): 500 x 250 x 120 (60 + 60) cm
✓ Masă totală maximă autorizată 14 tone (masă proprie 3,5 tone)
✓ Omologare pentru circulaţia pe drumurile publice
✓ Sistem de frânare hidraulică sau pneumatică (la alegere)
✓ 2 axe tandem cu suspensii pe foi de arc
✓ Roţi recondiţionate de 435/50 R19.5
✓ Apărători de noroi zincate
✓ Obloane cu sistem de deschidere sus / jos
✓ Scară de inspecţie
✓ Lumini de drum
✓ Cutie zincată pentru scule
✓ Mini-rezervor de apă curată pentru spălat pe mâini
✓ Gură de descarcă posterioară pentru cereale
✓ Şnec posterior pentru descărcare cereale

Opţional, obloane zincate sau din aluminiu, roată de rezervă.
