# Research: SEAP BIDDING ASSISTANT (IDEA-077)
Date: 2026-04-16

## Description
Ajuta firme sa liciteze pe SEAP. 5.1M licitatii in DB + alerte + matchmaking. EUR 200-500/licitatie + 5% success fee.

## Market Size
- Romania public procurement: billions EUR/year through SEAP/SICAP
- Thousands of SMEs need help navigating SEAP procedures
- Growing market as EU funds increase procurement volume

## Our Advantage
- 5.1M tenders in database (historical + active)
- Automated matching: company profile -> relevant tenders
- LLM-powered document drafting (technical + financial proposals)
- Email alerts already running (EBRD procurement monitor as proof)
- Can scale without proportional headcount (automation)

## Market Validated?
YES. 8+ competitors found. ForFuture charges 330-990 RON/2yr for basic monitoring. Full consulting is custom/premium. SEAP system fees 200-1200 RON for authorities.

## Price Point
- Monitoring: EUR 50/mo (alerts + matching)
- Bid preparation: EUR 200-500/tender (docs + submission)
- Success fee: 5% on won contracts
- Premium: EUR 300/mo (monitoring + 2 bids/mo included)

## Risk
- Legal liability if bid fails due to documentation errors
- Need domain expertise per sector (construction vs IT vs medical)
- Competitors have years of relationships with clients
- Regulatory changes can break automation

## Recommendation
GO with caution. Start with monitoring + alerts (low risk), add bid preparation later. The 5.1M tender database is a massive asset. LLM document drafting is the differentiator.
