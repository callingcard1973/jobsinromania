## RECOMMENDED: KDP + IngramSpark + Books.by (all three)
- **KDP** = Amazon discovery (biggest market)
- **IngramSpark** = bookstores + libraries
- **Books.by** = direct sales (highest margin, keep customer data)

## Amazon KDP Step-by-Step

### 1. Create KDP Account
- Go to kdp.amazon.com
- Sign in with Amazon account (or create one)
- Add tax info (W-8BEN for non-US) and bank account for payments

### 2. Prepare Files
- **Interior PDF:** KDP-compliant (embedded fonts, correct trim + margins)
- **Cover PDF:** Use KDP Cover Creator OR upload full wrap (front+spine+back)
- KDP provides a cover template calculator based on page count + trim size

### 3. Create New Paperback
- Click "Create" → "Paperback"
- Fill: Title, subtitle, author, description, 7 keywords, 3 categories
- Upload interior PDF + cover PDF
- Choose: B&W or Color, White or Cream paper, Trim size

### 4. Set Price
- **Minimum price** = printing_cost / 0.60
- Our 200-page B&W 8.5x11: $3.40 print → min $5.67 list → set $29.99-$39.99

### 5. Review & Publish
- Use KDP Previewer to check formatting
- Order proof copy ($3.40 + shipping) before going live
- Publish → live on Amazon in 72 hours

**Verdict:** B&W for directories. Color only for premium/photo catalogs.

## Profit Per Book

### B&W 8.5x11, 200 pages, priced at $29.99
- Print cost: $4.40
- KDP royalty (60%): $29.99 × 0.60 = $17.99
- **Your profit: $17.99 - $4.40 = $13.59 per book**

### B&W 6x9, 300 pages, priced at $24.99
- Print cost: $4.60
- KDP royalty: $24.99 × 0.60 = $14.99
- **Your profit: $14.99 - $4.60 = $10.39 per book**

## Cover Requirements
- Full wrap: front + spine + back in one PDF
- Spine width = pages × 0.0025" (B&W) or × 0.002347" (color)
- Bleed: 0.125" on all edges
- KDP provides downloadable template with exact dimensions

## ISBN
- **Free from KDP** (KDP-assigned, Amazon-only distribution)
- **Buy your own** ($125 for 1 from Bowker, or use free ones from some countries)
- IngramSpark requires your own ISBN
