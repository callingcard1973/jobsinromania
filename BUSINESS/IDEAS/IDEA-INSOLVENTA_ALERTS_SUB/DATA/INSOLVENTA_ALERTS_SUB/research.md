# Research: INSOLVENTA ALERTS SUB (IDEA-059)
Date: 2026-04-16

## Description
Subscription service for insolvency alerts: 770K companies + 222K bankruptcies + ANAF live data. Daily email alerts to accountants, lawyers, debt collectors, suppliers. EUR 29-99/month.

## Market Size & Demand
- Romania: 770K+ companies in insolvency database, 222K bankruptcies tracked
- Target buyers: ~50K accountants, ~20K lawyers, ~5K debt collectors, thousands of suppliers
- RisCo.ro already monetizes this exact data (paid insolvency alerts)
- Termene.ro has freemium model with paid premium features
- ListaFirme.eu offers insolvency data with email delivery and API
- Coface/CreditReform charge EUR 1K+/year for enterprise monitoring
- Excellent SME Romania: 297 EUR/year for company certification
- RisCo API: minimum 100 RON/month (~EUR 20)

## Competitors Found
| Competitor | Focus | Pricing | Strengths |
|-----------|-------|---------|-----------|
| RisCo.ro | Insolvency alerts + company data | ~100 RON/mo+ (API); paid alerts | Established brand, API, daily Excel |
| Termene.ro | Court deadlines + company monitoring | Freemium + paid | Large user base, court integration |
| ListaFirme.eu | Company directory + insolvency | Free + premium | SEO traffic, simple interface |
| Coface Romania | Credit insurance + monitoring | EUR 1K+/year | Global brand, insurance bundled |
| CreditReform RO | Credit reports + solvency index | Per-report + subscription | EU-wide coverage, debt collection |
| Excellent SME | Company certification | 297 EUR/year | Trust badge for websites |

## Our Advantage
- 770K companies + 222K bankruptcies already in database (larger than publicly available on competitors)
- ANAF live integration adds tax-status monitoring (unique combination)
- Already have cifn.eu domain with insolvency content live
- Email infrastructure ready (6,300/day capacity)
- Can undercut RisCo/Termene on price while offering combined insolvency + ANAF data
- Node-RED dashboard already built for internal use

## Market Validated?
YES, STRONGLY. RisCo.ro is a profitable business built on exactly this model. Termene.ro has large traffic. ListaFirme.eu is well-known. Multiple competitors confirm strong demand from accountants, lawyers, and credit managers.

## Price Point
- EUR 29/month: 50 company watchlist + daily email alerts + basic search
- EUR 59/month: 200 company watchlist + ANAF tax status + CSV export
- EUR 99/month: Unlimited watchlist + API access + ANAF + insolvency + court data
- Annual: 2 months free (EUR 290/590/990 per year)

## Risk
- HIGH competition. RisCo and Termene.ro are entrenched with loyal users
- Trust barrier: accountants and lawyers are conservative, slow to switch providers
- Data accuracy must be 100% - any error in insolvency status destroys credibility
- Legal: must source data from official BPI (Buletinul Procedurilor de Insolventa) legally
- Support burden: professional users expect phone support and SLAs

## Recommendation
GO but differentiate. Do NOT try to clone RisCo. Instead: (1) Bundle insolvency + ANAF + TED procurement in one dashboard (see IDEA-047), (2) Price aggressively at EUR 29/month to capture price-sensitive solo accountants, (3) Launch on cifn.eu which already has domain authority. Start with free tier (5 companies monitored) to build user base, then convert to paid.
