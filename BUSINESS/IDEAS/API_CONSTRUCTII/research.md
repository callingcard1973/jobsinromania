# Research: API CONSTRUCTII (IDEA-014)
Date: 2026-04-16

## Description
API service providing structured data on 117K Romanian construction companies. Sell access to developers, recruiters, and B2B sales teams needing construction sector intelligence.

## Market Size & Demand
- Romania has 117K+ construction companies (our data)
- European B2B data API market dominated by players charging EUR 425-25,000/year
- Construction is Romania's fastest-growing sector, driving demand for company intelligence
- ListaFirme.ro gets significant traffic (top competitor in RO company data)
- Global B2B data provider market growing rapidly — Cognism, HitHorizons, Bold Data all expanding

## Competitors Found

| Competitor | What | Pricing | Threat |
|------------|------|---------|--------|
| ListaFirme.eu/ro | Full RO company database + API, credit-based | Annual packages (undisclosed) | HIGH — established, covers all sectors |
| Termene.ro | Company catalog + legal/financial data | Subscription | HIGH — strong brand in Romania |
| AlertaCUI.ro | Company monitoring + insolvency alerts + API | Monthly subscription | Medium — monitoring focus, not bulk data |
| RisCo.ro | Company risk reports + API | From 100 RON/month + VAT | Medium — risk-focused, not sector-specific |
| romanian-companies.eu | English-language RO company API | Credit-based | Medium — same data sources |
| HitHorizons | 80M+ European companies API | Enterprise pricing | Low — pan-European, not RO-specialized |
| Dun & Bradstreet | Global company data inc. RO construction | Enterprise (EUR 5K+/year) | Low — expensive, general |
| Bold Data (CompanyData.com) | 303M companies, API | From EUR 425/dataset | Low — global, not specialized |

## Our Advantage
- Construction-sector specialization (117K companies vs. generic databases)
- Already enriched with emails, phones, CAEN codes, financial data
- Can include ISC data (23K construction professionals) as premium layer
- Much cheaper to operate than general-purpose competitors
- Can bundle with recruitment services (unique cross-sell)

## Market Validated?
PARTIALLY — ListaFirme and Termene.ro prove demand for Romanian company data APIs. However, no competitor offers a construction-sector-specific API. The niche may be too narrow for standalone viability but works as an add-on to existing operations.

## Price Point
- Basic API access: EUR 50-100/month (100-500 queries/day)
- Pro API: EUR 200-500/month (unlimited queries + enriched data)
- Enterprise/bulk export: EUR 1,000-5,000 one-time
- Competitors: RisCo from 100 RON/month, ListaFirme annual packages, Bold Data from EUR 425

## Risk
- HIGH: Crowded market with established players (ListaFirme, Termene, RisCo)
- Data freshness requirement — must continuously update from ANAF/ONRC
- GDPR compliance needed for personal data (emails, phones)
- Low switching cost for customers — easy to move to another provider
- 117K companies is small vs. ListaFirme's full database

## Recommendation
**PARK** — The market exists but is crowded with well-funded competitors offering broader coverage. A construction-only API is too niche to justify standalone development. Better to monetize this data through recruitment campaigns (already doing) or bundle it into a broader data product. Revisit if a specific customer requests it.
