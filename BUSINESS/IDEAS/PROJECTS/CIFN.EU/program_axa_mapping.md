# Common Program Operațional and Axa Codes (Romania/EU)

| Code   | Full Name                                 | Meaning/Focus                                  |
|--------|-------------------------------------------|------------------------------------------------|
| PRNE   | Program Regional Nord-Est                 | Regional development, Nord-Est region           |
| PRNV   | Program Regional Nord-Vest                | Regional development, Nord-Vest region          |
| PRC    | Program Regional Centru                   | Regional development, Centru region             |
| PRSM   | Program Regional Sud Muntenia             | Regional development, Sud Muntenia region       |
| PRSVO  | Program Regional Sud-Vest Oltenia         | Regional development, Sud-Vest Oltenia region   |
| PTJ    | Program Tranziție Justă                   | Just Transition (coal regions, green economy)   |
| PS     | Program Sănătate                         | Health infrastructure and services              |
| POCU   | Program Operațional Capital Uman          | Human capital, education, employment            |
| POC    | Program Operațional Competitivitate        | Competitiveness, innovation, digitalization     |
| POR    | Program Operațional Regional              | Regional development (2014-2020)                |
| FSE    | Fondul Social European                    | Social inclusion, education, poverty reduction  |
| SDS    | Scoala dupa Scoala                       | After-school programs for children              |

# How to use
- If a project description contains one of these codes, tag it with the corresponding program/axis.
- For more granularity, you can expand the table with additional axes or domains as needed.
