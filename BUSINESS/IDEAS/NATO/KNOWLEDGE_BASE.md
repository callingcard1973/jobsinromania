## 7. NCAGE CODE APPLICATION

### What is NCAGE?
NATO Commercial and Government Entity code - 5 character ID required for all NATO/US procurement.

### Format for Romania
#***L (number + alphanumeric + letter, e.g., R1234L)

### How to Apply
1. Go to: https://eportal.nspa.nato.int/AC135Public/CageTool
2. Click "Request New"
3. Fill: Entity type, Country, Organization name, Address, Contact
4. Select purpose: "Register in SAM.gov"
5. Submit, confirm via email link

### Processing Time
- Routine: 10 calendar days
- Emergency: 3 business days

### Cost
FREE

### Romanian Contact
- Email: ncage@roncb.ro
- Website: https://bnc.defense.ro/ncage_en.html

### Critical Rules
- Organization name must EXACTLY match SAM.gov entity validation
- Address must be exact, mailable physical address
- Phone format: 00country codephone (e.g., 0040723456789)
- One NCAGE per entity only
