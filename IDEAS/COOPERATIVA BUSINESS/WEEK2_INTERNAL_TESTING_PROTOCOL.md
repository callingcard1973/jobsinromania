### Phase 3: Decision & Iteration (Day 1, Night)

**Decision Tree:**

```
IF average score ≥ 4.0 on all dimensions:
  → APPROVED - Use as-is for mass send
  
IF average score 3.5-4.0:
  → APPROVED WITH EDITS - Incorporate main feedback
  
IF average score < 3.5:
  → REJECTED - Redesign and re-test (delay launch 1-2 days)
  
IF conflicting feedback (some 5s, some 2s):
  → A/B TEST - Send both versions, split 25 hypermarkets each
```

**Example Feedback Incorporation:**

| Original | Feedback | Revision |
|----------|----------|----------|
| "Trial order?" | Too casual/forced | "Let's discuss pricing for your first shipment" |
| "[BUTTON] Schedule Call" | Intimidating for first contact | "[BUTTON] Request Pricing Quote" |
| 3 product categories | Overwhelm concern | Simplify: show TOP 3 items, mention "10+ more available" |

## A/B TEST OPTION (IF CONFLICTING FEEDBACK)

If testers are split (some prefer Version A, some prefer B):

```
Hypermarket Group (50 emails) → Split Test:
  - Version A: 25 emails (Subject: Direct value)
  - Version B: 25 emails (Subject: Brand story)

Measurement (7 days):
  - Open rate: Which subject wins?
  - Click rate: Which CTA converts?
  - Response rate: Which tone gets replies?
  
Winner → Scale to 605 diaspora contacts (Week 5)
```
