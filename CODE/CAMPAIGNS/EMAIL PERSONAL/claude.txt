This is about working with the master DB on laptop (PostgreSQL 18, port 5433, interjob_master, 208M companies).

Focus: email quality filtering, MX validation, contact curation from DB.

Key scripts:
- step1_db_filter.sql — DB-internal filter (DNC, bounced, bad format, quality_tier)
- step2_mx_check.py — async MX DNS validation for 802K unknowns

Quality tiers in master_emails:
- Tier 1 (210K): MX valid + specific address — best for campaigns
- Tier 2 (15K): MX valid + generic (info@, office@)
- Tier 3 (802K): MX unknown — step 2 resolves these
- Tier 4 (4K): DNC / bounced / bad format
