================================================================================
TROUBLESHOOTING
================================================================================

Connection refused?
  → Check PostgreSQL: psql -U tudor -h 127.0.0.1 -p 5433 interjob_master -c "SELECT 1"

Table not found?
  → Verify: \d companies_clean in psql

Test fails with "Duplicate key"?
  → Clean up: DELETE FROM companies_clean WHERE source = 'ANOFM_TEST'

Tests hang or timeout?
  → Kill idle: SELECT pg_terminate_backend(pid) FROM pg_stat_activity

Password auth failed?
  → Set: SET PGPASSWORD=tudor
  → Or use .pgpass file (Windows: %APPDATA%\postgresql\pgpass.conf)

Pool overflow?
  → Increase size in conftest.py: SimpleConnectionPool(5, 20, ...)
