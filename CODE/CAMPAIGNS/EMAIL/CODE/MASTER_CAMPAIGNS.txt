DATABASE SCHEMA (companies_clean)
=================================

id INT PRIMARY KEY
email TEXT (indexed)
name TEXT
city TEXT
sector TEXT (indexed) — format: CAMPAIGN_SECTOR (e.g., ANOFM_CONSTRUCTII, NORWAY_AGRICULTURE)
source TEXT (indexed) — format: CAMPAIGN_YEAR (e.g., ANOFM_2026, NORWAY_2026)
country CHAR(2)
lead_score INT
enriched_email VARCHAR(255)
enriched_phone VARCHAR(50)
... (27 total columns, see CLAUDE.md EMAIL section)

SEND LOGS (send_log)
====================

id INT PRIMARY KEY
sector TEXT — matches companies_clean.sector
sent INT — emails sent in this batch
bounced INT — hard bounces
complaints INT — FBL complaints
date DATE
status TEXT — 'sent', 'bounced', 'complained'

DNC (Do Not Contact)
====================

email TEXT PRIMARY KEY
reason TEXT
added_at TIMESTAMP
campaign_blocked_until TIMESTAMP (optional)
