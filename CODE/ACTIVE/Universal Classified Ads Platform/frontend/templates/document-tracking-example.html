PostHog Document Tracking Marks

HTML Attributes for Tracking:

  • data-posthog-mark - Unique identifier for the tracked element (required)
  • data-doc-type - Type of document (ad, user, media, message, review)
  • data-doc-id - Unique ID of the document
  • data-action - Action performed (viewed, clicked, created, etc.)
  • data-prop-* - Custom properties to track (prefix with data-prop-)

Mark Format:

doc_{doc_type}_{doc_id}_{action}

Example: doc_ad_12345_view, doc_ad_12345_contact

Tracked Events:

  • document_ad_viewed - Ad viewed in list/detail
  • document_ad_clicked - Ad link clicked
  • document_ad_contact - Contact button clicked
  • document_ad_shared - Ad shared
  • document_ad_flagged - Ad flagged
  • document_ad_created - New ad created
  • document_ad_edited - Ad edited
  • document_ad_deleted - Ad deleted
  • document_ad_search - Search performed
  • document_ad_filter - Filters applied
  • document_ad_visible_in_viewport - Ad became visible

JavaScript Tracking API:

tracker.trackDocumentView('ad', 12345, { category: 'vehicles' });
tracker.trackDocumentCreated('ad', 12346, { category: 'vehicles' });
tracker.trackDocumentEdited('ad', 12345, { fields_changed: ['price'] });
tracker.trackDocumentDeleted('ad', 12345);
tracker.trackDocumentSearch('ad', 'honda', 42);
tracker.trackDocumentFilter('ad', { category: 'vehicles', price: '<5000' }, 15);
