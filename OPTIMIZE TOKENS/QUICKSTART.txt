DAILY WORKFLOW
================================================================================

1. START A SESSION
   python session_manager.py start "implement feature X"

2. WORK (in Claude Code: /code)

3. MID-SESSION (at 50% context fill)
   /compact

4. END SESSION
   python session_manager.py end
   /clear

================================================================================
PLUGIN MANAGEMENT
================================================================================

Disable Playwright for research tasks:
   python plugin_toggle.py browser off

Re-enable after task:
   python plugin_toggle.py browser on

Check current state:
   python plugin_toggle.py status

Restore defaults:
   python plugin_toggle.py defaults

Savings:
  - Playwright: ~400-600 tokens/session
  - Simplifier: ~200 tokens/session

================================================================================
MONITORING & AUDITING
================================================================================

Check active session:
   python session_manager.py status

View today's logs:
   python session_manager.py log --tail 20

Export session data:
   python session_manager.py export

Audit CLAUDE.md files:
   python claude_md_lint.py --summary

Full bloat report:
   python claude_md_lint.py

================================================================================
SESSION HYGIENE RULES
================================================================================

1. Use Haiku for research, Sonnet for implementation
2. Disable Playwright when not scraping
3. Run /clear between unrelated tasks
4. Run /compact at 50% context fill
5. Keep CLAUDE.md files <= 50 lines
6. Track sessions with session_manager.py
7. No sessions >30 min without /clear
8. Export logs weekly for analysis

See RULES.md for full guidelines.

================================================================================
TROUBLESHOOTING
================================================================================

"Settings file not found"
   → Create ~/.claude/settings.json with:
   {
     "enabled_mcp_servers": ["mcp__plugin_playwright_playwright"],
     "disabled_mcp_servers": ["superpowers:code-simplifier"]
   }

"No logs found"
   → Run: python session_manager.py start "test"
   → Then: python session_manager.py end
   → Check: python session_manager.py log

Plugin toggle not working
   → Verify: echo %USERPROFILE%\.claude\settings.json
   → Restart Claude Code to reload settings
